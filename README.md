# Project 1 - Static Website Hosting on AWS
Created a modern static portfolio website using:
- HTML
- CSS
- JavaScript

Then I deployed it to AWS using Amazon S3 static website hosting.

## What I learned
- How a website is just a collection of files
- How to host static files on AWS S3
- How to enable static website hosting
- How to make a website publicly accessible
- How to deploy a website from my laptop to the internet

## Files in this project
- `index.html` - main website page
- `style.css` - styling for the website
- `script.js` - simple JavaScript file
